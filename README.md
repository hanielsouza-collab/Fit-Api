# FitManager API

## Objetivo

Sistema backend para gerenciamento de academia, permitindo cadastro e controle de alunos e planos.

## Integrante

Haniel Sousa

## Entidades

### Alunos
- id
- nome
- idade
- peso
- altura
- plano_id

### Planos
- id
- nome_plano
- duracao
- preco
- beneficios
- status

## Endpoints

### Alunos
- GET /api/alunos
- GET /api/alunos/:id
- POST /api/alunos
- PUT /api/alunos/:id
- DELETE /api/alunos/:id

### Planos
- GET /api/planos
- POST /api/planos
- DELETE /api/planos/:id

## Tecnologias

- Node.js
- Express
- MySQL
- JavaScript
- HTML
